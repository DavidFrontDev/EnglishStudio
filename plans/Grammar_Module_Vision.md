# Модуль «Грамматика» (область Учёба) — видение и каталог идей

**Создано:** 2026-06-03. Идейный (vision) документ: что вообще может войти в модуль и как это ложится в приложение. Детальные пофазные планы + разбивка на двух агентов (контракты → A/B) — позже, по образцу «Чтения» ([`Reading_Study_Module.md`](Reading_Study_Module.md)).
**Область:** Учёба (`AppArea.Study`), новый classlib `EnglishStudio.Modules.Grammar`, метка «Грамматика» (иконка 📐), `order` между «Чтением» и «Статистикой».

---

## 1. Видение

Полноценный интерактивный тренажёр **всей** английской грамматики: для каждой темы — живая теория-карточка → интерактивная практика → упражнения → тест, с интервальным повторением правил (SRS) и безлимитной AI-генерацией заданий и проверкой свободных ответов (Claude). Центр UX — **интерактивные карточки** (особенно «все времена»): не статичный учебник, а конструкторы форм, таймлайны, контраст-таблицы, разборы ошибок.

**Принципы:**
- **Теория → практика → проверка → повторение** в каждой теме (не «прочитал и забыл»).
- **Объём + глубина:** от A1 до C1/C2, с нюансами и «ложными друзьями».
- **Адаптивность:** диагностика уровня, слабые места → персональная практика из лога ошибок.
- **Двуязычность:** объяснения и подсказки по-русски, материал — английский.

## 3. Центральная механика

### 3.1 Интерактивные карточки (ядро)
- **Карточка темы:** сворачиваемые правила, форма с цветовой разметкой (подлежащее / вспомогательный / смысловой глагол / окончание), примеры с аудио, блок «частые ошибки», «сигнальные слова».
- **Конструктор форм (для времён/залога):** выбираешь *глагол + лицо/число + время + аспект + залог + тип (утв./отр./вопрос)* → приложение строит все формы с подсветкой и подписями. Неправильные глаголы тянутся из словаря.
- **Таймлайн времён:** визуальная ось (точка/отрезок/завершённость) — наглядно «когда какое время».
- **Контраст-карточки:** Present Perfect ↔ Past Simple, will ↔ going to, some ↔ any, make ↔ do, this ↔ that и т.п. — две колонки + «выбери правильное».
- **Шпаргалки/справочник:** сводные таблицы (сетка 12 времён, неправильные глаголы, предлоги, окончания) — для быстрого повторения и (опц.) печати/экспорта.

### 3.2 Практика → тест → повторение
- Каждая тема: набор упражнений (раздел 5) → мини-тест → результат с разбором.
- **SRS-повторение грамматики:** карточки-правила и «карточки-ошибки» уходят в FSRS-очередь; повтор = вспомнить правило / заполнить пропуск / исправить ошибку.
- **Лог ошибок → адресная практика:** что часто ошибаешь — то чаще и в более узких заданиях.

## 4. Полный каталог грамматики (контент)

> Группировка по разделам; каждый пункт = тема с теорией-карточкой + практикой + тестом. ★ — приоритет/флагман.

### 4.1 Глаголы и времена ★ (центр модуля)
- **12 времён** (Present/Past/Future × Simple/Continuous/Perfect/Perfect Continuous) + **Future-in-the-Past**. Для каждого: форма (утв./отр./вопрос), случаи употребления, сигнальные слова, таймлайн, контраст с соседними, типичные ошибки.
- **Виды глаголов:** правильные/неправильные, вспомогательные (be/do/have), смысловые, **стативные vs динамические**, переходные/непереходные, глаголы-связки.
- **Модальные глаголы** (can/could, may/might, must, shall/should, will/would, ought to, need, dare) + полумодальные (have to, be able to, used to, be going to): оттенки (способность/разрешение/обязанность/вероятность/совет/запрет), прошедшие формы, **модальные перфекты** (must have done, should have done).
- **Залог:** активный ↔ пассивный (все времена), get-passive, **каузатив** (have/get something done).
- **Косвенная речь** (утверждения/вопросы/команды, backshift, изменения времени/места/местоимений).
- **Герундий vs инфинитив:** глагольные модели (verb+ing / verb+to / оба со сменой смысла: remember/stop/try), глаголы восприятия, голый инфинитив.
- **Согласование времён**, **сослагательное наклонение** (I suggest he be…), wish/if only.
- **Глагол + предлог** (зависимые предлоги), различие фразовых и предложных глаголов.

### 4.2 Условные предложения и гипотетика ★
- **Zero / First / Second / Third / Mixed** conditionals; таймлайн «реальность ↔ нереальность».
- wish / if only, would rather, it's (high) time, as if / as though, unless / provided / as long as / in case; **инверсия в условных** (Had I known…).

### 4.3 Существительные
- Исчисляемые/неисчисляемые, ед./мн. (правильные + **неправильные множественные**), собирательные, составные, абстрактные/конкретные, собственные/нарицательные.
- Притяжательный падеж ('s vs of), двойной генитив.
- Существительное+существительное (атрибутивное), партитивы (a piece of…), квантификаторы (см. 4.8).

### 4.4 Артикли и определители ★
- a/an/the/нулевой — правила + сложные случаи (географические названия, учреждения, обобщения, уникальные объекты, устойчивые выражения).
- Определители: this/that/these/those, притяжательные, each/every/either/neither/both/all/no/none/another/(the)other.

### 4.5 Местоимения
- Личные (субъект/объект), притяжательные (прил. + абсолютные), возвратные/усилительные, взаимные (each other/one another), указательные, **неопределённые** (some-/any-/every-/no- + body/thing/where), **относительные** (who/whom/whose/which/that, defining/non-defining), вопросительные, **«пустые» it / there**, one/ones как заместители.

### 4.6 Прилагательные и наречия
- **Порядок прилагательных** (OSASCOMP), степени сравнения (правильные/неправильные, as…as, the…the…), градуируемые/неградуируемые + модификаторы (very/absolutely/quite), причастия-прилагательные (-ed/-ing: bored/boring).
- Наречия: типы (образа действия/места/времени/частоты/степени), **позиция в предложении**, степени сравнения наречий, too/enough, so/such.

### 4.7 Предлоги
- Времени (at/on/in/by/until/for/since), места, движения/направления; **зависимые предлоги** (depend on, good at…); предложные обороты; предлог в конце вопроса/относительного предложения.

### 4.8 Союзы и квантификаторы
- Сочинительные (FANBOYS), подчинительные, **парные** (both…and, either…or, neither…nor, not only…but also), вводные/связующие наречия (however, therefore, moreover).
- Квантификаторы: much/many, few/a few, little/a little, some/any, a lot of/lots of/plenty of, all/most/none.

### 4.9 Числительные
- Количественные/порядковые, **даты/годы/время**, дроби/десятичные/проценты, большие числа, телефонные номера, чтение математики, приблизительность (dozens/hundreds of).

### 4.10 Структура предложения, придаточные, инверсия ★
- Порядок слов (SVO), **построение вопросов** (yes/no, wh-, вопрос к подлежащему, **разделительные** tag questions, косвенные вопросы, отрицательные вопросы).
- **Придаточные:** относительные (defining/non-defining, **reduced relatives**), дополнительные (noun clauses), обстоятельственные (времени/причины/цели/результата/уступки/условия), **причастные обороты**.
- **Инверсия** (Never have I…, Not only…but also, Hardly…when, So/Such…that, условная инверсия).
- **Cleft-предложения** (It is X that…, What… is…), **эллипсис и замещение** (so do I, neither/nor), **согласование подлежащего и сказуемого**.

### 4.11 Обороты и конструкции (то, что «и т.д.»)
- there is/are; have/get sth done; used to / be used to / get used to; would для прошлых привычек; «too/enough … to»; цель/результат (so…that, such…that, in order to, so as to); сравнительные конструкции; короткие ответы и so/neither-согласие; question tags.

### 4.12 Пунктуация и механика (легко)
- Заглавные буквы, запятые (в т.ч. Oxford comma), апострофы, дефис/тире, кавычки; правила правописания (удвоение согласной, -y→-ies, -ing); **различия British vs American** (грамматика и написание).

### 4.13 Функциональная грамматика (сквозные «как сказать X»)
- Выражение будущего; прошлые привычки; гипотезы; обязанность/разрешение/запрет; просьбы/предложения/советы — кросс-обзоры, связывающие несколько тем вокруг коммуникативной задачи.

## 5. Типы упражнений (движок практики)

- **Fill-in-the-blank** — вписать форму: «She ___ (go) to school» → goes.
- **Множественный выбор** — выбрать время/артикль/предлог.
- **Трансформация** — переписать: активный↔пассивный, прямая↔косвенная речь, объединить относительным придаточным, переделать условное, инверсия.
- **Исправление ошибки** — найти и починить.
- **Порядок слов / сборка предложения** — перетащить токены.
- **Сопоставление (matching)** — правило↔пример, время↔употребление, глагол↔предлог.
- **Cloze (связный текст с пропусками)** — несколько форм в абзаце.
- **Дрилл спряжения** — впечатать все формы глагола в заданном времени.
- **Свободная продукция** ★ — написать своё предложение с целевой структурой → **Claude проверяет** корректность + объясняет (как оценка Writing).
- Доп.: пропуск с выпадающим списком, достроить tag-question, выбрать «лишнее», аудио-диктант формы (TTS → вписать).

Объективные типы (fill/choice/matching/order) проверяются **локально** через переиспользованные answer-checker'ы IELTS Core; продукция и «почему» — через Claude.

---

## 6. Тесты и оценка

- **Тема-квиз:** смешанные типы, балл + разбор каждого ответа.
- **Диагностический/placement-тест:** оценка уровня → рекомендованные темы и стартовая точка.
- **Кумулятивные/смешанные тесты** (по разделу и по всему модулю), экзаменационный режим.
- **Mastery по темам:** % освоения, «слабые места», прогресс-бар по дереву тем.
- **Лог ошибок** → авто-генерация адресной практики + SRS-повторение именно ошибок.
- Интеграция в общую **«Статистику»** (сквозную): прогресс грамматики рядом с SRS/чтением.

---

## 7. AI (Claude) — что делает

- **Генерирует** упражнения любого типа по теме/уровню (безлимитная практика); кэширует по (тема, уровень, тип), как вопросы в Чтении.
- **Проверяет** свободные ответы/продукцию: верно/нет + **почему** (по-русски), с указанием правила.
- **Объясняет** по запросу: «почему here X, а не Y», разбирает конкретное предложение пользователя.
- **Генерирует примеры** в контексте и мини-диалоги на целевую структуру.
- **Персонализация:** из лога ошибок собирает индивидуальный набор заданий.
- Везде — **graceful offline:** при отсутствии Claude доступны засиженные карточки/упражнения и объективные тесты; генерация/проверка продукции — недоступны.

## 8. Геймификация и мотивация (опционально)

- Дерево/карта тем с разблокировкой; mastery-звёзды по темам.
- Дневные цели, стрик, «разминка дня» (5 случайных карточек из-за повторения).
- Достижения («освоил все времена», «100 предложений в пассиве»), уровни.

## 9. Модель данных (высокоуровнево)

```
GrammarSection  (Глаголы, Существительные, …)        — дерево разделов
 └ GrammarTopic (Present Perfect, Articles, …)        — тема: уровень CEFR, порядок, иконка
     ├ GrammarRule / GrammarCard  (теория-карточки: текст, форма, примеры, «ошибки», контраст)
     ├ GrammarExample             (EN + RU + аудио-ref; часть из словаря)
     └ GrammarExercise            (тип, формулировка, варианты, ключ/эталон, уровень, источник: Seed|Ai)
GrammarAttempt    (ответ пользователя, верно/балл, дата) — лог ошибок/история
GrammarProgress   (SRS-состояние карточки/правила: FSRS-поля)  — повторение; ОТДЕЛЬНАЯ таблица (не UserWordProgress), общий IFsrsScheduler
```
Контент: **ядро (времена/артикли/условные) — выверенный seed**, остальные разделы — **AI-догенерация** (`Source = Seed | Ai`). Свой `GrammarDbContext` (общий `dictionary.db`, своя история миграций). Глаголы/слова/аудио — ссылки в словарь, не дублировать.

## 10. Предлагаемый порядок реализации (фазы)

> Упражнения и тесты не отдельной фазой, а сразу вместе с каждой темой (движок практики — в G0).

| Фаза | Содержание |
|---|---|
| **G0 — Каркас** | classlib `Modules.Grammar`, `GrammarDbContext` + сущности (раздел 9), оболочка-модуль (Area=Study), навигация по дереву тем, **движок упражнений** (reuse answer-checkers) + базовый тест-раннер, SRS-хук для грамматики |
| **G1 — Времена ★ (по группам)** | флагман, по подфазам: **G1a Present** → **G1b Past** (+Future-in-the-Past) → **G1c Future**. В каждой: карточки + **конструктор форм** + таймлайны + контрасты + упражнения/тест. Времена — выверенный seed |
| **G2 — Глагольная система** | модальные (+перфекты), пассив/каузатив, косвенная речь, герундий/инфинитив, согласование времён, **условные** |
| **G3 — Имя** | существительные, артикли/определители ★, местоимения |
| **G4 — Признак и связки** | прилагательные/наречия, предлоги, союзы/квантификаторы, числительные |
| **G5 — Синтаксис** | структура предложения, вопросы, придаточные, инверсия, cleft/эллипсис, обороты/конструкции, пунктуация |
| **G6 — Сборка** | диагностический тест, mastery-дерево, лог-ошибок→адресная практика, геймификация, интеграция в «Статистику» |

Каждую фазу — как «Чтение»: отдельный детальный план + ШАГ 0 (контракты) + два агента (A=данные/контент/Claude/seed, B=UI карточек/практики).

## 11. Решения (закрыто 2026-06-03)
- **Источник ядрового контента — ✅ РЕШЕНО:** ядро (все времена, артикли, условные) — **выверенный seed** (embedded JSON, вычитан вручную); все остальные разделы — **AI-догенерация** через Claude с пометкой провенанса (`Source = Seed | Ai`). Сиденые темы read-only/доверенные, AI-темы помечаются и при желании вычитываются.
- **SRS — ✅ РЕШЕНО:** **отдельная таблица `GrammarProgress`** (грамматика ≠ слово), но переиспользует тот же `IFsrsScheduler` (FSRS-4.5). НЕ расширять полиморфный `UserWordProgress`.
- **Глубина G1 — ✅ РЕШЕНО:** времена осваиваем **по группам**, не все 12 сразу. Подфазы: **G1a Present** (Simple/Continuous/Perfect/Perfect Continuous) → **G1b Past** (+ Future-in-the-Past как часть прошлого нарратива) → **G1c Future**. Каждая группа = отдельный заход (план + ШАГ 0 + два агента), внутри — карточки + конструктор форм + таймлайны + контрасты + упражнения/тест.
- **Озвучка (остаётся как дефолт):** TTS из F4 (`System.Speech`) для произвольных предложений/форм; Oxford-аудио — только для словарных слов. Не блокирует старт.
